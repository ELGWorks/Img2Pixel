Img2Pixel, developed by ELGWorks, is a simple website-based application that turns uploaded images into pixelated-style image as an output. Privacy Policy

Effective Date: May 5, 2026

1. Overview

Img2Pixel is a simple web-based tool that allows users to upload images and convert them into a pixelated style. The application runs entirely in the user’s browser using HTML, CSS, and vanilla JavaScript.

2. Data Collection

Img2Pixel does not collect, store, or transmit any personal data.

  • No user accounts or logins are required
  • No images are stored on any server
  • No personal files or data are saved or shared

3. Image Processing

All image processing happens locally in your browser. Uploaded images are temporarily loaded into memory for conversion and are never sent to any server. Once the page is refreshed or closed, the data is automatically cleared.

4. Cookies and Tracking

This website does not use cookies, analytics tools, or tracking scripts.

5. Third-Party Services

This website may use third-party services such as Google Fonts to improve visual appearance. These services may collect basic technical information (such as browser type or IP address) according to their own privacy policies.

6. Data Security

Since all processing is done locally on your device, Img2Pixel does not store or transmit any user data. However, users are advised not to upload sensitive or private images to any online tool.

7. Changes to This Policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page.
